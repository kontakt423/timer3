# 🌙 SleepTimer

Ein moderner Android Sleep Timer mit stilvoll dunklem Design.

## Features
- ⏱ Timer-Einstellung: Stunden + Minuten, Schieberegler, Schnellauswahl
- 📱 Aktionen beim Timer-Ende:
  - WLAN ausschalten
  - Mobile Daten ausschalten
  - Ton stummschalten
  - Flugmodus aktivieren
  - Ultra-Energiesparmodus einschalten
  - Laufende Apps beenden
- 🔔 Foreground Service: Timer läuft auch bei gesperrtem Bildschirm
- 🎨 Modernes Night-Theme mit Compose Material3

## Tech Stack
- Kotlin, Jetpack Compose, Material3
- MVVM + StateFlow
- Foreground Service + WakeLock
- Mindestsystemanforderung: Android 8.0 (API 26)
